# Etch-a-Sketch
A browser based Etch-a-Sketch application built primarily with Javascript, HTML and CSS.

-Features:
Draw by hovering over the grid
Change the grid size (up to 100 × 100)
Clear the board
Random color mode

-Built With:
JavaScript
HTML
CSS

-What I Learned:
This project helped me practice DOM manipulation, event listeners, CSS Grid, and creating elements dynamically with JavaScript.
